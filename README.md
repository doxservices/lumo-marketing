# LUMO Marketing Static Site

Standalone static deployment for the LUMO creative studio concept.

## Files

- `index.html` — home page
- `portfolio.html` — portfolio spread
- `brands.html` — brands page
- `contact.html` — contact page
- `styles.css` — shared responsive styling
- `assets/*.svg` — lightweight deploy-safe visual panels

## Notes

The uploaded package included large PNG exports. The deployed version keeps the multi-page structure and larger spread direction, but uses SVG visual panels so the site can be committed cleanly through the available GitHub connector without broken binary image references.

Open `index.html` directly in a browser. No build step is required.
