# LUMO Studio — Behance Gallery Demo

Standalone static demo site.

## Files

- `index.html` — main site
- `assets/lumo-logo-blue.png` — blue logo for light backgrounds
- `assets/lumo-logo-white.png` — white logo for dark backgrounds
- `assets/favicon.png` — favicon

## Notes

The portfolio gallery uses public Behance project image URLs from `https://www.behance.net/MrWesley` rather than bundled local copies. The page needs internet access for those gallery thumbnails to load.

Open `index.html` directly in a browser. No build step and no external libraries.
